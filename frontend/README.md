Attendance System Frontend

This is the frontend application for the Attendance System. It includes dashboards for Admin, Lecturer, and Student roles, along with a public frontgate attendance check-in page.

## Installation

npm install

## Development

npm run dev

The application will be available at http://localhost:5173

The backend API should be running on http://localhost:8000

## Build

npm run build

## Features

- **Admin Dashboard**: User management, attendance reports, and system alerts
- **Lecturer Dashboard**: Session management, attendance tracking, and student performance
- **Student Dashboard**: Attendance history, upcoming sessions, and alerts
- **Frontgate Attendance**: Public page with facial recognition for attendance check-in
- **Role-based Authentication**: Separate login pages for Admin and Lecturer/Student
- **Protected Routes**: Routes are protected based on user role
- **Responsive Design**: Works on desktop, tablet, and mobile devices
